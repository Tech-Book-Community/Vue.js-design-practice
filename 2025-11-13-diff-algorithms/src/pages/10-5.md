# 循環結束後的檢查

<div class="mt-6 text-xl text-gray-300 mb-4">
完整的循環結束後處理邏輯：
</div>

<div class="bg-gray-800/50 rounded-lg p-5">

````md magic-move
```js {all}{lines:true}
while (oldStartIdx < oldEndIdx && newStartIdx <= newEndIdx) {
  // 省略
}

if (oldEndIdx < oldStartIdx && newStartIdx <= newEndIdx) {
  // 添加新節點
} else if (newEndIdx < newStartIdx && oldStartIdx <= oldEndIdx) {
  // 移除舊節點
}
```
```js {all|2-6|7-11}{lines:true}
// 循環結束後
if (oldEndIdx < oldStartIdx && newStartIdx <= newEndIdx) {
  // 添加新節點
  for (let i = newStartIdx; i <= newEndIdx; i++) {
    patch(null, newChildren[i], container, oldStartVNode.el)
  }
} else if (newEndIdx < newStartIdx && oldStartIdx <= oldEndIdx) {
  // 移除舊節點
  for (let i = oldStartIdx; i <= oldEndIdx; i++) {
    unmount(oldChildren[i])
  }
}
```
````

<!--
來看完整的迴圈結束後處理邏輯，把添加和移除都放在一起看。

[點擊] 第一個 if：如果舊節點的索引交叉了（oldEndIdx < oldStartIdx），但新節點還沒交叉，就代表要添加新節點。

[點擊] 第二個 else if：如果新節點的索引交叉了（newEndIdx < newStartIdx），但舊節點還沒交叉，就代表要移除舊節點。

用「交叉」來記憶：誰交叉了就代表誰處理完了，沒交叉的那邊還有剩餘。
-->
